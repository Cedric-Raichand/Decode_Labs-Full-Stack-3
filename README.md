# Industrial Training Kit - Database Integration Project

## Project Overview

This project is a backend application developed as part of an Industrial Training Full Stack Development program. It demonstrates database integration by connecting an Express.js backend application with MongoDB.

The project implements a user management system where data can be stored, retrieved, updated, and deleted using REST API endpoints.

## Features

- MongoDB database integration
- User database schema design
- Create users (POST)
- Retrieve users (GET)
- Update users (PUT)
- Delete users (DELETE)
- JSON request and response handling
- Environment variable configuration using dotenv

## Technologies Used

- Node.js
- Express.js
- MongoDB
- Mongoose
- JavaScript
